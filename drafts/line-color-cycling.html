Line-color cycling

tags: matplotlib, line plots, mpltools

Sequential curves

Sometimes it's useful to plot a series of lines and change the color of those lines in sequence.

import numpy as np
import matplotlib.pyplot as plt

n_lines = 10
x = np.linspace(0, 10)

phase_shift = np.linspace(0, np.pi, n_lines)
color_idx = np.linspace(0, 1, n_lines)
for i, shift in zip(color_idx, phase_shift):
    plt.plot(x, np.sin(x - shift), color=plt.cm.cool(i))

plt.show()
